Neuron Culture Embodiment Plan Overview

  • Requirements
    • Both optical spike input and output
    • High interface bandwidth ~ 100,000 bits/s
    • High neuron count ~ 100,000 neurons
  • Culture Development
    • Optogenetic rat cortical neuron line
    • Utilizing genetically encoded opsins and biosensors, eg. GCaMP, ChRger2
  • Optical Interface
    • CMOS sensor for neuron spike output (Guo)
    • microOLED display for neuron spike input
  • Virtual Enviroment
    • Malmo Minecraft Enviroment
    • Neuron Culture training performed with periodic signal rewards and random signal punishments (Kagan)
    • Improving predictions rewarded
    • Worsening predictions punished

Optical Neural Interface (ONI)

  • ONI v0
    • First version of interface will be read only
    • Sony OV2311 CMOS sensor for neural output
  • Final Interface Hardware Rough Specs
    • Sony OV2311 CMOS sensor for neural output
    • Sony OLED ECX337AF display for neural input
    • Convex lens projects neural optogenetic emissions and projects display light onto neurons
    • Beam splitting prism enables sensor and display to work on same neuron area
